# Sage Website Design (SWD) 🌐

Studio site + portfolio + live client previews. Hosted free on GitHub Pages.

🔗 **Studio site (your portfolio / visitekaartje):** https://sagek17.github.io/swd/

## Structure
- `index.html` — **Sage Website Design studio site** (your portfolio link for email/signature).
- `work/` — your real projects, hosted for live preview thumbnails:
  - `work/archidec/` — Archi & Dec Ghana (links out to live https://archidec.netlify.app/)
  - `work/ghana-house/` — Ghana House (Extensions & Barber Shop)
  - `work/members-club/` — Friday Members Club · Rigtersbleek
  - `work/specta/` — Specta brand site
- `clients/<name>/` — client previews to send to prospects (e.g. `clients/deters-hairstyling/`).
- `template-demo/` — reusable service-business demo (not featured on the site).
- `assets/` — logo files: `swd-logo-dark.svg`, `swd-logo-light.svg`, `swd-mark.svg`.

## Upgrade path
Buy `sagewebsitedesign.nl` → add as a free custom domain in Settings → Pages →
set up `sage@sagewebsitedesign.nl` (Zoho Mail free or a registrar mailbox).
